Learn Spring Boot

Spring Boot can be weird to learn if you haven't encountered MVC (Models Views Controllers) application structures before, luckily there are some great tutorials around to allow beginners to start using Spring Boot.

Lots of great learning resources can be found here on the Learn section of Spring's website. They have plenty of tutorials for different topics and quick start guides for common project types.

While researching, I have also come across a series of video tutorials on YouTube by a channel called LaunchCode. They are really useful for beginners as the content is covered slowly, adding functionality step-by-step while explaining concepts. I have compiled the series into an unlisted YouTube playlist which can be accessed here.
